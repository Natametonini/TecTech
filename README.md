# Projeto Garra Robô com Arduino para Separação de Lixo

## Descrição
Este projeto tem como objetivo o desenvolvimento de uma garra robô inteligente capaz de identificar e diferenciar itens recicláveis, não recicláveis, orgânicos e não orgânicos. Utilizando um **Arduino UNO**, sensores e atuadores, a garra é capaz de separar os diferentes tipos de lixo e movê-los para os devidos compartimentos. O sistema visa promover a eficiência na reciclagem e o gerenciamento adequado de resíduos, utilizando tecnologia simples e acessível.

## Objetivo
- Criar um sistema automatizado para reconhecimento e separação de resíduos.
- Utilizar Arduino UNO como controlador principal.
- Desenvolver a garra robô para manipular os objetos.
